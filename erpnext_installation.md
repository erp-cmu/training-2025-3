## Docker Method
- `git clone -b hrms https://github.com/nnnpooh/frappe_docker.git`
- `docker compose -f docker-compose-hrms.yaml up -d`
- Remove Docker containers (if needed)
    - `docker compose -f docker-compose-hrms.yaml down -v`
